요소접근방법

    웹문서 HTML의 모든 태그를 객체형식으로 표현 
    (html 태그 (body,h1,pre,span)를 js에선 객체라고함)

DOM을 이용한 태그 접근 

    id로 접근하기 : document.getElementById("아이디 값");
    id가 일치하는 태그를 가져옴


    class로 접근하기 : document.getElementByClassName("class이름값");
    class가 일치하는 태그를 가져옴
    class이름이 일치하는태그가많다면 모두가져와서 배열형태로 바꿔서 전달

    name으로 접근하기 : document.getElementByName("name 값");
    name과 일치하는 태그를  모두가져와서 배열형태로 바꿔서 전달

  DOM을 이용해서 태그를 접근하고 접근한 태그에는 추가하거나 수정하거나 
  제거하는 역활을 해주는 속성이 있음 

  textContent : 태그에서 지식 태그의 모든 텍스트 내용을 보여줌 
                 HTML 태그를 무시하고 가져오거나 설정할 떄 사용
                외부 공격을 방지할 때 사용 
                innerText와 같지만 다른방식으로 작동 쓰임은 같음 

  innerText :  태그에서 자식태그의 모든 태그 내용을 보여줌 
  HTML 태그를 무시하고 텍스트만 가져오거나 설정할 때 사용 
  css새로설정해서 적용하거나 
  추가적인 스타일 변경이필요한 경우 사용 
  외부 공격방지는 하지못함
